# 크롬 익스텐션 설치 가이드

## 빠른 설치 (3단계)

### 1단계: 크롬 확장 프로그램 페이지 열기

크롬 브라우저 주소창에 입력:
```
chrome://extensions/
```

### 2단계: 개발자 모드 활성화

우측 상단의 **개발자 모드** 토글 스위치를 클릭하여 활성화합니다.

### 3단계: 익스텐션 로드

1. **압축해제된 확장 프로그램을 로드합니다** 버튼 클릭
2. 파일 탐색기에서 이 폴더(`flex-addon`) 선택
3. **폴더 선택** 버튼 클릭

완료! 이제 Flex 근무 기록 페이지에서 자동으로 작동합니다.

## 사용 방법

1. https://flex.team/time-tracking/my-work-record 접속
2. 로그인
3. 우측 상단에 보라색 카드로 **이번 주 총 근무시간** 표시됨

## 동작 확인

### 정상 작동 시
```
┌─────────────────────────────────┐
│   이번 주 총 근무시간             │
│      33시간 06분                 │
│                                  │
│  누적: 23시간 20분 + 오늘: 9시간 46분  │
└─────────────────────────────────┘
```

### 문제 발생 시

**아무것도 표시되지 않는 경우:**
1. F12를 눌러 개발자 도구 열기
2. Console 탭 선택
3. `[Flex 익스텐션]` 메시지 확인
4. 에러가 있다면 스크린샷을 찍어서 문의

**시간이 이상한 경우:**
- 페이지 새로고침 (F5)
- 익스텐션 재설치
- Chrome 재시작

## 익스텐션 제거 방법

1. `chrome://extensions/` 접속
2. "Flex 근무시간 합산" 찾기
3. **제거** 버튼 클릭

## 업데이트 방법

1. 새 버전 파일로 교체
2. `chrome://extensions/` 에서 익스텐션의 **새로고침** 버튼 클릭

## 문의

문제가 발생하면 다음 정보와 함께 문의해주세요:
- 크롬 버전
- 에러 메시지 (개발자 도구 Console)
- 스크린샷
