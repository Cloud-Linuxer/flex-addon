# Flex 근무시간 합산 익스텐션

Flex 근무 기록 페이지에서 **이번 주 총 근무시간(오늘 포함)**을 자동으로 계산하여 표시하는 크롬 익스텐션입니다.

## 기능

- ✅ 이번 주 누적 근무시간 + 오늘 근무 중인 시간 자동 합산
- ✅ 시각적으로 보기 쉬운 UI로 총 근무시간 표시
- ✅ 페이지 로드 시 자동으로 계산 및 표시
- ✅ 세부 내역 표시 (누적 시간 + 오늘 시간)

## 설치 방법

### 1. 익스텐션 파일 준비

이 폴더에 있는 파일들:
- `manifest.json` - 익스텐션 설정
- `content.js` - 실행 스크립트
- `styles.css` - UI 스타일

### 2. 아이콘 파일 준비 (선택사항)

아이콘이 없어도 익스텐션은 정상 작동합니다. 원하시면 아래 크기의 PNG 이미지를 추가하세요:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

아이콘을 추가하지 않으려면 `manifest.json`에서 `icons` 섹션을 삭제하세요.

### 3. 크롬에 익스텐션 로드

1. 크롬 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 폴더 선택 (`flex-addon`)

## 사용 방법

1. https://flex.team/time-tracking/my-work-record 페이지 접속
2. 로그인 후 자동으로 우측 상단에 **이번 주 총 근무시간** 표시됩니다

### 표시 예시

```
┌─────────────────────────────────┐
│   이번 주 총 근무시간             │
│      33시간 06분                 │
│                                  │
│  누적: 23시간 20분 + 오늘: 9시간 46분  │
└─────────────────────────────────┘
```

## 계산 방식

- **이번 주 누적 시간**: 우측 상단에 표시된 시간 (예: 23:20)
- **오늘 근무 중인 시간**: 좌측 상단에 표시된 "근무중 X시간 Y분"
- **총 근무시간**: 누적 시간 + 오늘 시간

## 문제 해결

### 시간이 표시되지 않아요
- 페이지를 새로고침 해보세요 (F5)
- 개발자 도구(F12) → Console 탭에서 에러 확인
- `[Flex 익스텐션]` 로그 확인

### 시간 계산이 이상해요
- 페이지 구조가 변경되었을 수 있습니다
- GitHub Issues에 스크린샷과 함께 제보해주세요

## 개발 정보

### 파일 구조
```
flex-addon/
├── manifest.json    # 익스텐션 설정
├── content.js       # 메인 로직
├── styles.css       # UI 스타일
├── README.md        # 문서
└── icon*.png        # 아이콘 (선택사항)
```

### 주요 기능
- `parseTimeToMinutes()`: 다양한 시간 형식 파싱
- `formatMinutesToTime()`: 분을 "X시간 Y분" 형식으로 변환
- `calculateAndDisplayTotalTime()`: 시간 계산 및 표시
- MutationObserver: 페이지 변경 자동 감지
